## Report New Issues

If you encounter a problem not listed here:

1. Check the [Issues](https://github.com/asi-alliance/asi-chain/issues) section for similar reports
2. If not found, open a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Browser/environment information
   - Screenshots if applicable
