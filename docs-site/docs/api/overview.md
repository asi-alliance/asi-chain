# API Overview

ASI Chain provides comprehensive APIs for blockchain interaction.

## API Types
- **gRPC**: High-performance binary protocol
- **HTTP REST**: Web-friendly JSON API
- **WebSocket**: Real-time events

## Base URLs
- gRPC: `grpc://api.asi-chain.io:40402`
- HTTP: `https://api.asi-chain.io`
- WebSocket: `wss://api.asi-chain.io/ws`
