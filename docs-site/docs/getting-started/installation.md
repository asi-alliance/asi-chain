# Installation

Get started with ASI Chain development.

> **📢 Latest:** Documentation now auto-deploys via GitHub Actions!

## Requirements
- Docker 20.x or higher
- 8GB RAM minimum
- 20GB free disk space

## Quick Start
```bash
# Clone the repository
git clone https://github.com/asi-alliance/asi-chain.git

# Start local node
docker-compose up -d
```
